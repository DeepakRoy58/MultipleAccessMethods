TDMA Working

In TDMA, users share the same frequency but take turns in different time slots. Only one user transmits per slot. The cursor below sweeps through time; each highlighted block represents a user's slot.

  • Bandwidth: shared.
  • Time: divided into slots per frame.
  • Good for: bursty traffic, scheduled access.

Time Focus

TDMA Overview

Uses a "Round Robin" approach where each user gets the full frequency for a tiny slice of time.

  • Concept: Sharing via Time Segregation.
  • Key Example: 2G GSM Cellular Networks.
  • Advantage: No interference from simultaneous users.
  • Disadvantage: Precise timing/sync is critical.
Frequency Focus

FDMA Overview

Divides the total bandwidth into separate "lanes" or channels. Each user stays in their own lane.

  • Concept: Sharing via Spectral Segregation.
  • Key Example: Old Analog Radio, Satellite TV.
  • Advantage: Simple hardware; no timing sync needed.
  • Disadvantage: Inefficient if a user isn't talking (waste).
Code Focus

CDMA Overview

Users send data at the same time on the same frequency, but "encoded" with unique mathematical keys.

  • Concept: Sharing via Mathematical Segregation.
  • Key Example: 3G Networks, GPS.
  • Advantage: Highly secure and high capacity.
  • Disadvantage: Complex receivers; Near-Far problem.
