# Multiple Access Techniques Visualizer

### Project Overview
This is an interactive educational tool designed for Electronics and Communication Engineering (ECE) students and learners. It provides a visual way to understand how different users share a single communication channel using various multiplexing methods.

### Learning Purpose
The goal of this project is to simplify complex wireless communication concepts. Instead of looking at static diagrams in a textbook, users can see how data flows in real-time. 

This project was developed as a learning exercise. The logic, animations, and descriptions have been researched and collected from various educational resources to create a helpful, all-in-one learning tool.

---

### Features
- Real-time Visuals: Dynamic animations showing how users are separated by time, frequency, or codes.
- Interactive Controls: Users can adjust the number of participants and even provide custom names.
- Technique Selection: Switch between TDMA, FDMA, and CDMA instantly to compare them.
- Pause and Play: Ability to stop the animation to study a specific frame.
- Simplified Explanations: Each technique comes with a clear description and key characteristics.

---

### Techniques Covered

Time Division Multiple Access (TDMA)
Users share the same frequency but are assigned specific time slots. Only one user transmits at a time. It is like people taking turns to speak in a meeting.

Frequency Division Multiple Access (FDMA)
The total bandwidth is divided into separate frequency bands. Every user has their own "lane" and can transmit at the same time as others. It is like a multi-lane highway.

Code Division Multiple Access (CDMA)
All users transmit at the same time and on the same frequency. However, each user is assigned a unique mathematical code to distinguish their signal from others. It is like people in a room speaking different languages at once.

### Technologies Used
- HTML5: For the structure of the application.
- CSS3: For the layout, styling, and animations.
- JavaScript: For the simulation logic and user interface interactions.
- Browser-Based: No installation or backend is required. It runs entirely in the web browser.

---

### How to Use
1. Open the index.html file in a web browser.
2. Enter the Number of Users in the input field.
3. Optionally, enter custom names separated by commas in the User Names field.
4. Choose a Technique (TDMA, FDMA, or CDMA) from the dropdown menu.
5. Click the Show Working button to generate the visualization.
